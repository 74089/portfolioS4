Architectuur

Tijdens de ideation fase ga ik hier nog niks voor doen. Ik wil tijdens de ideation fase zo veel mogelijk afronden van de andere leerdoelen zodat ik in de implementation fase hier volledig op kan focussen. Ik wil dan, als het niet voor mijn groepje is, voor mezelf een website maken die gebruik maakt van een API en modulair is ingericht maken. De website is gebaseerd op een structuur diagram, user stories en research. Als dit alles in de website zit wordt het een 200%.

De student heeft een applicatie gerealiseerd met een module pattern. Externe libraries zijn gebruikt om de applicatie te ontwerpen. De applicatie is tot stand gekomen via user stories en een schets van de architectuur.

Ontwikkelomgeving

Ik heb al veel ervaring met werken in professionele werkomgevingen omdat ik al een MBO opleiding heb gedaan voor applicatie ontwikkelaar. Ik maak gebruik van Atom met een aantal packages. Atom heeft ook een ingebouwde Git feature die ik gebruik om te pushen, pullen en commiten. Voor communicatie met mijn groepje gebruiken we Microsoft Teams en Whatsapp, voor communicatie met mijn leraren is Whatsapp verwisseld door Canvas.

De student heeft een onderbouwde (front-end) ontwikkelomgeving samengesteld en laat zien op de hoogte te zijn van de in het werkveld gangbare tools
