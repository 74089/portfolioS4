Architectuur

Arduino Breadboard draadjes tekenen

Bij ons proftaak project maken we een website waarbij kinderen probleem oplossend leren denken en ze maken kennis met de basis van Arduino.

Het gedeelte dat ik programmeer is gebaseerd op onze user story: “Als leerling wil ik zien hoe ik de draadjes op mijn Arduino moet plaatsen zodat ik niet”. Ik heb dit berijkt door op een afbeelding van een breadboard een canvas te zetten. Op het canvas kan je rechte lijntjes trekken.

Getekende lijntjes op een canvas

Mogelijke uitbreidingen zouden nog zijn dat de lijntjes alleen vanuit zwarte puntjes naar zwarte puntjes kunnen getrokken worden en een button die de laatste lijn en alle lijntjes weghaalt.

GitHub Repo

Dit is het eerste project dat ik gemaakt heb waarbij ik het modulair probeerde te maken, ik heb dit gekregen door de hele tijd te denken: "Gaat dit ergens problemen veroorzaken als ik het ergens anders implementeer". Ook heb ik de basis van canvas geleerd, hierin had ik ook nog geen ervaring.

Arduino SVG

De vorige versie hiervan was via canvas, als ik het met canvas had gedaan was er geen makkelijke mogelijkheid om de lijntjes op de juiste punten te laten snappen.

Feedback 1

Misschien zou je naar SVG kunnen kijken en het via SVG punten aan elkaar verbinden.

Feedback 2

Op het moment dat ik deze feedback kreeg had ik een Arduino bordje als SVG maar nog geen functionaliteit. Ik had ook al geoefend met lijntjes trekken op een grid. https://jsfiddle.net/u8qy19ms/3/# Maar dit is later niet gebruikt.

Je kan makkelijk SVG manipuleren net zoals een canvas via een library die Raphaël heet.

Feedback 3

Op het moment dat ik deze feedback kreeg had ik het probleem dat de lijntjes niet op de punt van je muis kwamen terwijl dat wel de bedoeling was.

In de SVG had ik een viewbox gedefinieerd en die bleek mijn muis coördinaten helemaal verkeerd te pakken. Gelukkig wist mijn leraar welke woorden hij in stack overflow moest typen om een makkelijk antwoord te krijgen. https://stackoverflow.com/questions/10298658/mouse-position-inside-autoscaled-svg

Nadat dat geïmplementeerd was werkte de lijntjes hoe ik wilde maar de lijntjes kwamen achter het Arduino SVG’tje, dit is opgelost door de SVG in een functie te zetten en die aan te roepen voor de layer waar je al de lijntjes op tekent.

Wat heb ik er geleerd

• Met SVG’s werken

• SVG viewbox

• Paar functies in de Raphaël JS library

Conclusie

Er zijn meestal meerdere antwoorden mogelijk als het om programmeren gaat, maar om jezel het makkelijkste te maken moet je de simpelste weg nemen. De simpelste weg in dit geval was met een paar bochten waar ik nog geen ervaring mee had. Deze bochten, met name SVG en Raphaël, waren een zeer leerzame ervaring en het is goed dat ik weet dat ze bestaan en dat ik weet hoe ze gebruikt worden als ik ze nog een keer nodig heb.

De student heeft een applicatie gerealiseerd met een module pattern. Externe libraries zijn gebruikt om de applicatie te ontwerpen. De applicatie is tot stand gekomen via user stories en een schets van de architectuur.

Ontwikkelomgeving

Editor & version control tool

Zelf gebruik ik de editor Atom, ik heb hier al bijna 3 jaar ervaring mee. Tijdens mijn vorige opleiding mochten we onze eigen editor gebruiken en toen kreeg ik Atom geadviseerd, door er meer en meer mee te werken begon ik het fijner en fijner te vinden. Atom heeft een ingebouwde Git functie waarbij je kunt inloggen met je GitHub Account en vervolgens makkelijk kunt pushen, pullen, commiten en van branch kunt wisselen. Er zijn ook heel veel packages gemaakt door de community (een package in Atom is het zelfde als een package in VScode). Een van die packages heet Git-log, als je het opent mag je een van je repos kiezen om vervolgens een overzichtelijke lijn van alle branches en commits te zien. Door deze package ben ik gestopt met GitKraken te gebruiken en volledig overgestapt op allen Atom. Ik heb ook nog veel meer packages die hier te zien zijn.

De student heeft een onderbouwde (front-end) ontwikkelomgeving samengesteld en laat zien op de hoogte te zijn van de in het werkveld gangbare tools
